[[analysis-ngram-tokenfilter]]
=== NGram Token Filter

A token filter of type `nGram`.

The following are settings that can be set for a `nGram` token filter
type:

[cols="<,<",options="header",]
|============================
|Setting |Description
|`min_gram` |Defaults to `1`.
|`max_gram` |Defaults to `2`.
|============================
